# React Day 4

## Overview
This project is part of a React learning series, focusing on Day 4 concepts and exercises.

## Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Basic React knowledge

## Installation
```bash
npm install
```

## Getting Started
```bash
npm start
```
